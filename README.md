# S-Shop E-commerce Website

Một website e-commerce được xây dựng bằng HTML, CSS và JavaScript thuần.

## Tính năng

- Giao diện responsive
- Tìm kiếm sản phẩm
- Giỏ hàng
- Đăng nhập/Đăng ký
- Quản lý tài khoản
- Thông báo

## Công nghệ sử dụng

- HTML5
- CSS3
- JavaScript (ES6+)
- FontAwesome Icons
- Google Fonts

## Cài đặt và chạy

1. Clone repository:
```bash
git clone https://github.com/your-username/s-shop.git
cd s-shop
```

2. Cài đặt dependencies:
```bash
npm install
```

3. Chạy development server:
```bash
npm run dev
```

4. Mở trình duyệt và truy cập: `http://localhost:3000`

## Deploy lên Vercel

Website này đã được cấu hình để deploy lên Vercel. Chỉ cần:

1. Push code lên GitHub
2. Kết nối repository với Vercel
3. Deploy tự động
